# Student Management 

## Overview
Simple Java Spring Boot starter project.
- Spring Boot (Web, Thymeleaf, Data JPA)
- H2 in-memory database
- CRUD operations for Student entity

## Run
1. Build: `mvn clean package`
2. Run: `mvn spring-boot:run`
3. Open: http://localhost:8080/students
4. H2 console: http://localhost:8080/h2-console (JDBC URL: jdbc:h2:mem:studentsdb)
